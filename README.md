# tmpgb
Game Boy emulator written in C. This is a toy project to learn more about emulation and hardware.
WIP!

## Building

### Prerequisites
- SDL2

### Linux
Install SDL2 and build with `make`.

## Usage
```
tmpgb [options] <rom>
Options:
  -b <bootrom>  Start with executing the bootrom
  -d            Start in debug mode
```
